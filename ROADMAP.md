# Huey World - 개선 로드맵 🦊

## 🔧 버그 수정 (우선순위 높음)

### 1. 연결 상태 표시 개선 ⭐
- **문제**: "Status: Offline" 텍스트가 화면에 계속 남아있음
- **원인**: `this.connText` 라이프사이클 관리 문제
- **해결**: 연결 성공 시 텍스트 제거 또는 상태 업데이트
- **파일**: `main.scene.js`
- [ ] 수정 완료

### 2. 닉네임 동기화 ⭐⭐
- **문제**: 접속 시 다른 플레이어가 "Unknown"으로 표시됨
- **원인**: Join 핸드셰이크에서 닉네임이 즉시 전달되지 않음
- **해결**: `join_game` 이벤트에 닉네임 포함, 서버에서 브로드캐스트 시 닉네임 포함
- **파일**: `socket.manager.js`, `server.py`
- [ ] 수정 완료

### 3. 플레이어 가시성 ⭐⭐⭐
- **문제**: 게스트가 호스트를 못 보거나 그 반대 현상
- **원인**: `current_players` 이벤트 핸들링 미흡
- **해결**: 접속 시 모든 기존 플레이어 정보 순회하여 추가
- **파일**: `socket.manager.js`, `server.py`
- [ ] 수정 완료
